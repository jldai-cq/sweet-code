下载git
在本地电脑某个目录文件夹上右键打开git：进入该文件夹路径

1、绑定用户
git config --global user.name "jianglin"
git config --global user.email "daijl_cq@163.com"

2、为github账户设置SSH key，git使用加密算法rsa
生成ssh key：$ ssh-keygen -t rsa -C "daijl_cq@163.com"
将生成的ssh key添加到github账户中

3、上传本地项目到存储库repository
git init
git add .
git commit -m "first commit"
git remote add origin git@github.com:shengyp/Agri-RelExIns.git
git remote -v 查看已连接的存储库
git push -u origin main 
输入密码  Enter passphrase for key '/c/Users/代江林/.ssh/id_rsa': 0604

报错信息： ! [rejected]        main -> main (fetch first)
error: failed to push some refs to 'github.com:shengyp/Agri-RelExIns.git'
hint: Updates were rejected because the remote contains work that you do not
hint: have locally. This is usually caused by another repository pushing to
hint: the same ref. If you want to integrate the remote changes, use
hint: 'git pull' before pushing again.
hint: See the 'Note about fast-forwards' in 'git push --help' for details.

报错原因：远程仓库已经更新，本地仓库未pull更新

解决方法1-强制覆盖远程仓库：git push -u origin main --force

解决方法2-拉取远程更改并合并：git pull origin main   
         将远程的更改合并到你的本地分支  git push -u origin main
